Overview

This section discusses the included examples of the FMIFlux.jl library. So you can execute them on your machine and get detailed information about all of the steps. If you require further information about the function calls, see library functions section. For more information related to the setup and simulation of an FMU see FMI.jl library.

The examples are:

  • Simple hybrid CS: Showing how to train a Neural CS FMU.
  • simple hybrid ME: Showing how to train a Neural ME FMU.
  • advanced hybrid ME: Showing how to train an advanced Neural ME FMU.
